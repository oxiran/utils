# @oxiran/utils

Shared utils of my project.

## Installation

### Using npm
```shell
npm install @oxiran/utils -S
```

### Using yarn
```shell
yarn add @oxiran/utils
```
### Using CDN
```javascript
<script src="https://unpkg.com/@oxiran/utils@1.2.0/dist/utils.min.js"></script>
```
